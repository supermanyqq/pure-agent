# Pure Agent Core

Core runtime used by [Pure Agent CLI](https://www.npmjs.com/package/@pure-agent/cli).

This package is published so the CLI can resolve its runtime dependency. It is
not currently offered as a stable standalone SDK.

Install the CLI:

```bash
npm install -g @pure-agent/cli
```

Source and documentation: [supermanyqq/pure-agent](https://github.com/supermanyqq/pure-agent)
